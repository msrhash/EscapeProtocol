# Escape Protocol

## Overview
Escape Protocol is a 2-player cooperative obstacle platformer game inspired by puzzle-adventure mechanics. Players must work together to navigate through traps, collect key pieces , and overcome challenges that require coordination and timing.

The game is inspired by classic co-op puzzle games like *Fireboy and Watergirl*, but includes an original storyline and a final boss encounter that connects to the game’s hidden lore.

---

## Objective
The main objective of the game is to:
- Get through each stage
- Collect pieces
- Successfully escape the final stage

---

## Game Features
### Co-op Gameplay
- Designed for 2 players 
- Requires teamwork to collect pieces and pass obstacles

### Puzzle Mechanics
- Timing-based traps
- Environmental switches and triggers
- Coordination-based progression

### Boss Fight
- Final stage includes a boss battle
- Combines movement, timing, and pattern recognition

### Core Experience
- Platforming challenges
- Progressive difficulty
- Team coordination-focused gameplay

---

- C#, Unity

## How to Play
1. Start the game
2. Explore the environment
3. Get through the obstacle
4. Collect pieces puzzles to unlock next stages
5. Reach the final escape point
